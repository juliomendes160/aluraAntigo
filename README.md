# Angular CLI

[Angular CLI](https://angular.io/cli)

**Comandos:**

- `npm install` <!--node_modules-->
- `ng [v | version]`
- `ng new nome-project`
- `cd nome-project`
- `ng serve`
- `ng [g c | generate component] nome-component`
- `ng [g i | generate interface] nome-interface`
- `ng [g s | generate service] nome-service`
